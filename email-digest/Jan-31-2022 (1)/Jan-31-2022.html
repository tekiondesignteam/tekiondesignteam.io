RO Dashboard
 
The RO Dashboard provides Service Advisors and Technicians with clear visibility into RO activity. It provides a clear column-based view of RO’s in different statuses, allowing you to visualize the RO pipeline, identify tasks to be completed, and take action as and when needed.

Parts & Inventory Details Page - Revamp
 
One of the main Pain points for our user was that, due to current Information Hierarchy, our users weren’t able to consume the data correctly. The increase in cognitive load made it hard for the user to rememberer which section of the screen had which info. To solve this, we restructured the Information hierarchy, and grouped it based on each data point functionality. This way it will make the user consume the info efficiently and also give them an extra ability to re-arrange the data based on their preference!

Rockstars behind the project:
Prasheed CP, Shyamashree Banerjee

Physical Inventory
 
Physical inventory is a new feature where it helps audit the parts count and the availability of the parts in the dealership. Every dealership performs a physical inventory task once a year or half-yearly or quarterly to evaluate the inventory value of the dealership. Physical inventory is the evaluation of the entire inventory whereas perforated inventory is the evaluation of a select quantity of bins.

Custom Report Builder
 
The goal of this project was to create a custom report for the user based on the primary data that he/she provides. It is then scheduled as per requirement. The primary data source is the highlight for this report builder and is super flexible to use and generate reports. Currently this entire process of creating template reports is created by default on our platform is not scheduled/or customizable. This new revoulutionary revamp helps users to create their own reports, exactly the way they want to.
