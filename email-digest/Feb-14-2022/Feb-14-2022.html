Report Access Permission
 
This feature provides the user with the option to give access to specific users for viewing or editing a report. It helps them in avoiding unwanted confusion and prevent from users from viewing or editing the report that they are not supposed to. We also provide them with the option to either add a single user or a user group. The permission can also given to the user for multiple reports at once.
