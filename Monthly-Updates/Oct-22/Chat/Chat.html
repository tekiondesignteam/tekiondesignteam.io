Chat - October Design Updates
 
 
Contact Centre: Transfer Chat Enhancements
Figma Prototype
 
 
Chat can be transferred to an internal Call Center Agent or Dealership. For agents, users can select from the list of online agents and send the transfer request with the message(optional) which helps the other agent while picking up the chat. The request is sent with a timer of 1min. In dealership transfer, the user will send a chat request to the dealership and not to the selected agent.
