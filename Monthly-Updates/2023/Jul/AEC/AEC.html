Acura/Honda - VDP math box pricing use cases
Figma File
We have examined two main scenarios for the VDP (Vehicle Detail Page) math box section. The first scenario arises when a user enters a trade-in value that surpasses the vehicle's price. An informational message with a 'contact dealer CTA' is displayed in such instances while turning off the pricing section and concealing the price itself. The second use case occurs when the system cannot retrieve the pricing information. In such cases, we present the net price of the vehicle along with a 'contact dealer CTA,' allowing the user to acquire additional details from the chosen dealership regarding their purchase process.

Acura - Email Templates
Figma File
We have crafted scalable email templates that accommodate various scenarios, including displaying a single vehicle or a list of three or more vehicles in a template. Inchcape: Sharing vehicle details via email
Figma File
This feature enables users to share vehicle details with up to five individuals via email. The user interface is designed to resemble Gmail, where email addresses are added as chips. Once five addresses are added, adding more addresses is restricted, and an error message will be shown. Alternatively, users can copy a link to share vehicle details through other communication channels.
