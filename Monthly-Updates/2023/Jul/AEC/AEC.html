In July, the team was working on the next iteration of the configuration comparison, tablet and mobile layouts for previously completed and approved designs, and the newest concept of the Express Configurator. Details follow:
Configuration Comparison Next Iterations
Figma Prototype
Configuration Comparison allows users to add and remove items directly from the comparison page. A user also can highlight differences, re-organize an order of displayed configurations and change the view layout (collapse sections) to make it easier to read and compare.
Also, a comparison page is designed to accommodate up to 5 vehicles simultaneously and a virtually unlimited number of compared items. This is also included in the tablet and mobile versions of the configuration page.
Dmitri Bekasov
Sr. Design Manager
Expert Configurator Concept
Figma Prototype
The Expert Configurator makes it possible to build a comprehensive configuration in seconds using a command line and unified code system. A user could configure a vehicle of any complexity and proceed to the checkout without waiting for the system to update the front-end experience. This functionality is meant for expert users familiar with configuration options looking for a rapid and effective way to place large fleet orders.

Sr. Design Manager
