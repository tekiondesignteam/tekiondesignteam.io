DRP - July Design Updates
Cadillac - Preferred Dealer Selection
Figma File
Currently, in the VSR flow, the user can view inventory based on the ZIP code and radius set when the user lands on the VSR page. By adding this feature, the user can select their specific dealership by searching the dealership name or ZIP code. Users can select a dealership at any point in time during their exploration on VSR and Configurator.

Cadillac - Share Design Code
Figma File
Enhanced the share design code interaction for the Cadillac configurator summary page. Now, when the user completes configuring a vehicle to their preferences, a design code is generated. User can now choose from four options, copy the code, download a pdf of the summary, add to favorites or share the design to friends. This new feature enables the user to quickly save or share their personalized vehicle design.

CarBravo - Updated Protection Plans
Figma File
We have updated the protection plan detail view for the DRP platform. The detail view will now include mileage along with number of months. Adding mileage promotes responsible vehicle usage, encouraging customers to be mindful of their driving behavior and maintain their vehicles properly. This can lead to reduced maintenance costs and lower risks for the service provider, translating into potentially lower premiums for customers.
