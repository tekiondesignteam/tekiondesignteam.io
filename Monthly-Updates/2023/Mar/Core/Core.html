Reports Repository
Figma Link
Currently, we create canned reports for specific Dealer Partners as per their requests. If another Dealer Partner asks for the exact same report, we would have to separately create the report for them. There is no functionality to replicate/share already created reports for other Dealers. Reports repository allows Users to have a centralised repository for all reports, share and replicate reports as required.

Implementation Automation
Figma Link
As a part of onboarding dealerships Integration Teams at Tekion needed a way to quickly copy different module setups into the new dealerships. Right now this process is manual and time consuming thereby affecting dealer onboarding negatively. In order to solve for that, we made a separate app called Implementation Automation which allows Tekion integration specialists to migrate settings from one dealership to newer dealership at the click of a button. Additionally, it allows the ISMs to take backups of the settings as well, so that they can restore the setting whenever anything goes wrong.
The Sequence Settings app was also integrated into Implementation Automation for ISMs to manage their work from one single place.
Sougata Nair
Product Designer Lead
 
 
Sequence Settings
Figma Link
As a part of onboarding dealerships Integration Teams at Tekion needed a way to manage all the sequence numbers (RO, SO, Customer#, etc) such that they can reset them after the data has been migrated into Tekion. Additionally we put minimum, maximum and prefixes to these sequences. In the coming releases we will have more improvements to this sequencing, so that they cannot be repeated easily.
Sougata Nair
Product Designer Lead

When a large group of people are exchanging messages in a channel, it becomes difficult to keep a track of all of them. It would be easy if agents get to know of all the messages that are addressed to them. This helps certain users in focussing only on those messages where s/he was addressed.
