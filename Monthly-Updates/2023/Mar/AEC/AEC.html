Acura/Honda - March Design Updates
 
 
Secure Checkout Flow
Figma File
 
 
To improve the vehicle purchase journey of the users, we have designed a new and updated secure checkout process for Acura. The latest flow includes a new set of lead forms, trade-ins, credit application, Co-buyer application, Schedule delivery & in-store pickup, Switch vehicle (by customer and dealer separate), and notifications. The entire experience has been revamped and tried to make it simpler and easier for users to access and travel through the longer flow much more efficiently. We are continuously enhancing the design to accommodate all different use cases and address all the possibilities.

Honda - My Garage
Figma File
 
 
MyGarage is a new function/flow that will be plugged into the Honda/Acura shopping flow. It helps the user in keeping track of appointments & orders. These allow the user to keep track of their actions on the platform. Overall, MyGarage is a valuable tool that can streamline the process of managing appointments and orders for Honda/Acura shoppers and owners.
