RRG: Warranty Management
Figma Design Link
 
 
This feature will allow to the user remove parts from the inventory and manage warranty claim and credit warranty invoices from OEM. Track and manage warranty claims on SO in the system that respect Renault warranty flow. As a warehouse manager, I need to be able to manage warranty claims in the system. Warranty SO = warranty invoice = warranty claim. All this is the same object, in the requirement text below different terms (out of this 3) can be used.

Health Check : Parts
Figma Design Link
 
 
User gets to know about set up issues from dealerships, service managers or themselves. There is a lot of to & fro communication between dealership & ISMs in order to find & confirm about the set up errors & rectify them. There is no central location to view set up or other errors across modules. If they are unable to resolve it, they connect to support person. If the support person is not able to help he/she assigns the support ticket to engineering team. All the related steps taken are explicitly mentioned by the relevant persons in the support ticket until the issue is resolved then, the ticket is closed.
 
Shyamashree Banerjee
Product Design Lead
