Live Chat Notification Centre
Figma Prototype
 
 
Since multiple messages can be notified through the same icon, it is becoming difficult for the user to identify the notifications. A notification centre will help users identify the chats and prioritise to reply accordingly. Users can see where did the message come from, who has sent it, what is the message, and at what time.

Live chat templates - Phase 2
Figma Prototype
 
 
A web chat template is a pre-designed framework for building a chat interface on a website. It allows agents to quickly create and integrate a chat feature on their website without starting from scratch. Web chat templates can be customised and bulk edit as needed to match the specific needs of a web chat.
