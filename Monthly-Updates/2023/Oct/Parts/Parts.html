Parts Design Updates - October 2023
Create PO From SO
Figma Design Link
What?
For a part not in stock, users need to create OEM, Vendor and Miscellaneous PO screen while staying within SO page.
Why?
Currently, in order to create a PO, users first create a SOR, then either open a new tab or exit out of SO page to navigate to the PO screen and then create a PO. This is a lengthy procedure and adds multiple clicks to the process.
How?
Ability to create, reissue and open receiving a purchase order from within a sales order.
Priya Singh
Senior Product Designer
Bulk actions to manage multiple sales orders
Figma Design Link
What?
To enable the users to manage multiple sales orders at once
Why?
Currently, users have to perform document level actions one by one on every sales order. This slows down the user due to multiple click
How?
Having bulk action to manage multiple sales orders increase the efficiency of the user and save time.
