Acura ZDX - My Appointments
Figma File
This central hub allows users to oversee and coordinate all test drives or showroom visit appointments. Our intuitive and user-friendly interface enables users to effortlessly schedule appointments, providing a convenient way to reserve their slots with the dealership. Furthermore, users enjoy the flexibility to adjust their appointments should their plans change, enhancing convenience and adaptability within the experience. In summary, this feature greatly elevates the user experience by delivering a streamlined platform for personalized interactions with the dealership.

Acura ZDX - My Order
Figma File
The Acura ZDX, My Order feature provides users with a seamless way to oversee their current inventory vehicles and the status of custom orders. Users can effortlessly access vehicle information and checkout statuses with a simple click or tap. A dedicated Order Details page has been established for a comprehensive overview, granting users access to crucial information such as price summaries and configurations. As the central hub for all purchase-related information, the My Order feature offers users a convenient and comprehensive solution.

Acura ZDX - Custom Order
Figma File
Once the user finishes configuring their vehicle, we offer two options. First, the user can view the available inventory if the exact configuration is in stock. We can place a custom order through our platform if the desired configuration isn't in stock. Users can choose their preferred dealership and complete the payment for the custom order. The entire process can be tracked conveniently within the 'My Account' section.
