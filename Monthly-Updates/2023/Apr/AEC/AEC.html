Acura/Honda - April Design Updates
 
 
Secure Checkout Flow
Figma File
 
 
Updated Acura secure checkout experience. The updates include improvements to various aspects of the checkout process, such as lead forms, trade-in, credit application, co-applicant, document upload, signature, and payment flows. The team took into account the different use cases and error scenarios that users might encounter during the checkout process and worked to create a seamless and user-friendly experience that would accommodate all of these scenarios. Iterated designs throughout the entire experience, including changes and improvements to the design to improve interactions and reduce the number of clicks needed to complete the process.

CarBravo - Save Search
Figma File
 
 
The Save Search feature is an important addition to the CarBravo as it allows users to save their specific search criteria and receive notifications when new listings match their saved search. This feature not only saves time for users who frequently search for specific vehicles, but it also enhances the user experience by providing a personalized and proactive approach to car shopping. Additionally, the Save Search feature can increase engagement and retention rates for CarBravo as users are more likely to return to the platform to check their saved searches and view new listings. Overall, the Save Search feature provides value to both users and the CarBravo platform, making it a crucial feature to implement.
