Report Builder : Regional/Enterprise Workspace Enhancements
Figma Prototype
In the new enterprise construct, the following design interventions have been introduced to incorporate a regional workspace.

1. Region Selection Dropdown: Added a region selection dropdown to the new Report and Edit Report title modal. Now, when creating or editing a report, you can specify the region it belongs to.
2. Region Column in Report Builder: In the Report Builder view report and edit report flows, a new "Region" column has been introduced, making it easier to identify the region associated with each report.
3. Enhanced Report Titles: Report titles now clearly indicate the region and dealership they belong to, providing instant context.

These updates enhance our Report Builder's functionality, ensuring reports align with regional requirements.

Associate Product Designer
