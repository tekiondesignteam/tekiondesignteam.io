TAP CRM - Order Forms SKU’s Configuration Change
Figma Prototype
Context : In Tap CRM, under Order Forms, the Solution Managers Creates or Drafts an Order Form which is then marked as “Contract Drafted” Status. The SM’s may change some SKU’s configurations in that order form. The order form is sent to Solution Directors for their Approval and marked as “Pending Approval”

Problem Statement : While approving the order form, the solution directors were not able to see what changes have been made at the SKU’s configuration by the Solution Managers.

Solution : Added an extra step, where the Solution Directors can review what all fields values has been changed for each of the SKU’s. Within that they can also Approve/Decline those changes.
