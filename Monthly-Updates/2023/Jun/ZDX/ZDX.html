ZDX - My Account
Figma
The Acura ZDX My Account feature offers convenient management of both current inventory vehicles and custom order vehicle status. With just one click or tap, users can access vehicles and checkout statuses. They can also view and modify existing appointments, as well as review past appointments. For comprehensive information, a dedicated Order Details page has been created, providing users with access to essential details like price summaries and configurations. The My Account feature serves as the central hub for users to access all purchase-related information.

ZDX - VSR
Figma
The VSR Filter Section has undergone notable transformations, introducing the following enhancements:
 
  • Revamped the left navigation filter section with a sleek and contemporary design, improving user accessibility.
  • Utilized advanced data visualization techniques to present visually appealing and informative filtered results
  • Incorporated micro-interactions throughout the left navigation filter section to increase user engagement and usability.
  • Streamlined the filtering process based on valuable user feedback, resulting in fewer steps and improved efficiency.
These changes collectively enhance the overall user experience by facilitating more straightforward navigation, efficient data filtering, and informed decision-making within the application.

ZDX - VDP
Figma
We have made targeted improvements to the ZDX model's Vehicle Display Page (VDP), focusing on enhancing its visual representation. Users can access all vehicle images through the 'View Gallery' button. The user interface has been refined, incorporating interactive elements and highlighting the features section. Additionally, users can easily add accessories, compare vehicles, initiate trade-ins, and schedule test drives within the VDP section. These enhancements aim to provide a comprehensive and streamlined experience for users exploring the ZDX model.

ZDX - Schedule appointments
Figma
The appointment scheduling feature allows users to conveniently book appointments with the dealership. The part offers two options: A test Drive and a Showroom Visit, allowing users to select their preferred appointment type. They can then choose a suitable date and time from the available options. The user-friendly interface ensures easy navigation and adjustments during the booking process.
Furthermore, users can reschedule appointments if their plans change, providing convenience and adaptability. This feature significantly enhances the overall user experience by offering a streamlined platform for personalized interactions with the dealership.
