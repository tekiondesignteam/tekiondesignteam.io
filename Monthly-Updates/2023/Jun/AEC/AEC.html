Aurora Project —
June 2023 Design Updates
In June, the team continued improving and refining the configurator experiences, including designing new sidebar options selectors, search, configuration comparison and many other areas. Details follow.
Configuration Comparison
Figma Prototype
Configuration Comparison is meant to make it possible to compare various configurations in detail. A user can select up to 5 vehicles to compare specifications and configuration options side by side using a browser on any device, including smartphones. The first iteration of the comparison design has been approved for development and implementation. The next iteration will introduce the next set of features and reflect customer feedback from the first implementation.

Universal Modal Design
Figma Prototype
To bring the configurator design closer to the design system's look and feel, the team performed a series of visual improvements to the floating modal design pattern. Making these changes can be important when the configurator will be adapted to the longer words in European languages.

Sidebar Selectors and Options Comparison
Figma Prototype
Sidebar selectors are an important improvement as they allow abandoning bulky and somewhat confusing More Info floating panels. Instead, a user stays in the current flow and makes a selection based on a detailed description of each available option. A future proposed design also allows users to compare each of the possibilities a series line by line, similar to a comparing configuration.

Sidebar Configurator Search
Figma Prototype
Sidebar configuration search is the next generation of search functionality. A user can now perform real-time refinement of the displayed configuration options by entering keywords, price ranges or even order codes. It allows a user to quickly and effectively configure known items.
