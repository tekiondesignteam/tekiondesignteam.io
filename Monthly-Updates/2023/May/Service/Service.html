Scheduler : Assign Mode and SA Assignment
Figma Link
Currently all calendars in the system used for planning and allocating resources (like service advisors, technician etc) are independent and do not provide end-to-end visibility. With the new scheduling calendar we provide the users with this much needed visibility by showcasing all resources in one calendar view and thus helping them plan and allocate these resources better.
This workflow includes how the assign mode and service advisors assignment works within the scheduler.

Studio - Metadata Export & Import
Figma Link
Feature that allows users to export metadata (excluding actual data) from one workspace and import it into another. The objective is to streamline workspace setup and ensure consistency across multiple environments. By selectively exporting metadata and providing an import functionality, users can easily transfer workspace configurations, saving time and effort.
