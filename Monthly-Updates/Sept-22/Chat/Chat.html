Chat Platform - September Design Updates
 
 
 
 
Contact Centre: Admin Screens
 
Sneha A | Figma Prototype
 
 
Admins should be able to set up skills and skill groups, where they can add or edit primary and secondary skills which is assigned to contact centre agents. For each agent, admin can specify the number of chats that can be handled by them at a given point of time. They can also add or edit Account notes which is specific to Dealer / OEM / Brand.
 
 
 
Multi channel live chat integration - CRM/Chat - iMessage business
 
Manali More | Figma Prototype
 
 
Customer should be able to switch from live chat to iMessage business while he is active on live chat. Design the verbatim flow of different scenarios on the customer screen. iMessage business will help users to create leads and customers can schedule an appointments through the same app.
