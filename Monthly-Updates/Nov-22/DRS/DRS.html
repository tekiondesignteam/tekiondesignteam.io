Vehicle Unavailable - Lead form
Figma File
 
 
In case any vehicle that the user searched for is unavailable, we show a highlighted display banner where the user can request further details from the dealer via a lead form or can configure the vehicle and send a custom request. We will also display a list of similar vehicles that are currently available for the user to browse through, which in turn helps the user engage.
