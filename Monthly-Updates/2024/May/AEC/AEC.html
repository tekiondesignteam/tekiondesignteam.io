My Appointment GD screens
Figma File
After the user submits all their information and clicks on the final CTA to book an appointment, the dealership checks the availability of that slot internally. Then, the following two cases can happen: if multiple users try for the same slot, only one succeeds while the others see an "unavailable" message; if the dealership adjusts the slot concurrently, users see a "Try Again" message.
