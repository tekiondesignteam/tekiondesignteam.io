Parts
Ability to modify SORs
Figma Design Link
What?
Now the SOR can be edited even after creation. Following operations are allowed:
1. Add a part
  1. Delete a part
  2. Increase sales quantity
  3. Decrease sales quantity

Why?
Currently when a SOR is created, if user wants to add more parts for customer or change contact information there is no way to change it. Also, with every change in SO/RO, a new SOR is created.

Bulk Actions in SOR ( List & detailed view )
Figma Design Link
What?
We want to increase user efficiency by giving them bulk actions for multiple parts in a SOR document so that they do not have to go through each and every SOR to perform these actions.

Why?
If there are multiple parts on a SOR hold the user must move each to on hand which is time consuming.

How?
Enabling the user to perform bulk actions on different SORs ( list view ) and different parts within an SOR (detailed view )

Enhancements to Orders Section in Parts RO
Figma Design Link
What?
To be ordered and PO created sections have been introduced. First section show the parts that have been requested in an SOR and the second section shows that POs created and their respective part details.

Why?
In order to make this consistent with sales order and to improve visibility to the orders ( SOR & PO ) placed with a parts RO, a new tab has been introduced.

How?
The orders tab has been redesigned completely and made consistent with sales order.
