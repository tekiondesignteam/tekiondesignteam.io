PayPal payment method flow in Custom Order
Figma File
As planned by AHM, we have incorporated PayPal as an option for the users to make the deposit/reservation payment in the custom order process. We have enhanced the entire payment method flow to accommodate the new payment method along with card and ACH flow. For better visualization, we have also covered all use cases like payment failure, success, processing, etc. The mobile screens were also modified to make them consistent with the secure checkout designs so that the same components can be reused.
