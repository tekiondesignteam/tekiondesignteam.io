Parts
Multi warehouse manager dashboard
Figma Design Link
What?
For a user with superuser permission to manage all warehouses, there has to be a way for the user to track different metrics happening through a dashboard(No. of open PO, total no. of parts etc...). This dashboard should be flexible enough to display the metrics for multiple warehouses tagged to a dealership.

Why?
When the multi-warehouse feature gets build, it will be difficult for the user to switch between different warehouse to get the metrics for each warehouse one by one. By building this feature the user will be landing on a single page with all the necessary metrics populated for the user and take an action based on that.

Porsche - Modify & Cancel Appointment
Figma Design Link
What?
Based on Porsche’s feedback, we need a cancel appointment flow from emails which covers the following scenarios:

  • As a customer, I want to cancel the appointment I have just booked while I am still on the booking confirmation page
  • I want to cancel the appointment from my appointment-related emails - booking confirmation/reschedule/ other emails.
  • As a customer, I want to modify the appointment date and time for which I just got an email confirmation/ email reminder

Why?
  • Not having the ability to cancel an appointment from the booking confirmation page is a Porsche compliance issue.
  • Inability to cancel an appointment from the appointment emails - confirmation, reminder etc. in 2 or fewer steps
  • Modify appointments from emails, does not land the user in a modified appointment flow ( shows booking confirmation screen which is out of context, irrelevant step)

How?
We are providing the customer to cancel or reschedule the appointment by going through the Modify appointment flow from the email confirmation which is a 2-step solution and easily accessible for the user to cancel the appointment when required.
