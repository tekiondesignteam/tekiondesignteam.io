Reconditioning - Mechanical and Cosmetic Recon Workflows
 
Abhishek Adhikari | Figma Prototype
 
 
A dealer should be able to manage mechanical work and cosmetic work for Reconditioning. The dealers should be able to mark a vehicle as "Frontline ready". The dealer should also be able to create ROs from the "Waiting for recon" sheet.
Using this dashboard, the dealer will be able to manage vehicles for reconditioning, and also answer some key business questions such as:

"How many cars require cosmetic work"
"How many cars have dent work in progress"
"How many vehicles are frontline ready

Ask T - Search Bar Insights (Exploration)
 
Abhishek Adhikari | Figma Prototype
 
 
Ask T insights offers an high-level sneak peak into the various modules tailored for all personas ranging from dealers to sales managers to executive management.
Users can type in their query in natural language and Ask T responds with a summary of key data points which empowers users and eases visualising broad overview of their performance or trends.

Few query examples:
"How many new cars did I sell today?"
"What is my gross profit for this month?"
"What is the AR Balance for this month??"
